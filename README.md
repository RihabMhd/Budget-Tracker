# 💰 Budget Tracker

A web-based application designed to help users manage their finances efficiently by tracking income, expenses, and financial activities.

---

## 📌 Features

* ✅ User authentication (login/register)
* 💸 Track income and expenses
* 📊 Categorize transactions (service categories included)
* 📈 View recent transactions and analytics
* 📱 Responsive design for all devices
* ⚙️ Clean MVC structure with controllers and views

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS (Tailwind CSS), JavaScript
* **Backend:** Laravel
* **Database:** MySQL
* **Version Control:** Git & GitHub

---

## ⚡ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/RihabMhd/Budget_Tracker.git
   ```

2. Move to the project folder:

   ```bash
   cd Budget_Tracker
   ```

3. Set up your local server (e.g., Laragon, XAMPP)

4. Import the database (if provided)

5. Run the project in your browser:

   ```
   http://localhost/Budget_Tracker
   ```
